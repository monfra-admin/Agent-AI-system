# Anthropic Claude & Agents

**Updated 2025**: This directory contains examples and implementations using Anthropic's Claude API for building agents.

## Overview

Anthropic provides comprehensive resources for building effective agents with Claude, including Claude Skills and support for the Model Context Protocol (MCP).

## Resources

- [Anthropic Quickstarts](https://github.com/anthropics/anthropic-quickstarts)
- [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook/tree/main)
    - [Building Effective Agents Cookbook](https://github.com/anthropics/anthropic-cookbook/tree/main/patterns/agents)
- [Claude Skills](https://docs.anthropic.com/claude/docs/skills): New API for building agents with Claude (2025)
- [Model Context Protocol (MCP)](https://modelcontextprotocol.io/): Standard protocol for connecting AI models to external tools and data sources

## Key Features (2025)

- **Claude 3.5 Sonnet**: Optimized for agentic workflows, coding, and long-context tasks
- **Claude Skills**: API for building reusable agent capabilities
- **MCP Support**: Integration with Model Context Protocol for tool and context provisioning
- **Computer-Using Capabilities**: Claude-based computer-using agent features
