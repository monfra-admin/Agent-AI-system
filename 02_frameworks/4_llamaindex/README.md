# LlamaIndex

**Updated 2025**: LlamaIndex is a framework for building AI agents that can use external data sources, with strong integration for RAG-focused applications and data-centric agent systems.

## Overview
LlamaIndex is a framework for building AI agents that can use external data sources. It's particularly strong for:
- RAG-focused agent systems
- Data-centric agent architectures
- Multi-agent workflows with retrieval capabilities
- Integration with vector databases and knowledge bases
